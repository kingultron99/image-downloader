# Image Downloader
### by king_ultron99
built for D1a1v1e1

designed to grab the first 20 images from google and allow the user to select and save the images they want.

supports XLSX files with a list of queries they wish to search, alternatively you can search for individual terms.

built using React, tailwind and electron.
